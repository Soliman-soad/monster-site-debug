SessionStorage vs LocalStorage

SessionStorage is storage which expire its storage when the tab is over or reloaded. Where LocalStorage storage don't expire the data and collect data until its has been forced to remove. Session Storage objects can be accessed using the sessionStorage read-only property. The difference between sessionStorage and localStorage is that localStorage data does not expire, whereas sessionStorage data is cleared when the page session ends.

Block scope vs Global scope

A block is a piece of code inside a set of curly braces {} and groups code statements together.A variable exists inside or outside a block is known as BlockScope.If a variable is declared outside all functions or curly braces ({}), it exists in the global scope.

Event loop

A programming structure that continually tests for external events and calls the appropriate routines to handle them. An event loop is often the main loop in a program that typically waits for the user to trigger something.

Undefined

A variable that has not been assigned a value is of type undefined . A method or statement also returns undefined if the variable that is being evaluated does not have an assigned value. A function returns undefined if a value was not returned .
